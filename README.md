# web-form-login-management

Google Apps Script を TypeScript + clasp で管理する Web フォームログイン管理システム。

## 開発環境構築

1. Node.js をインストール
2. npm install
3. clasp login
4. clasp create --rootDir src --type standalone

## ディレクトリ構成

```
.
├ .clasp.json
├ README.md
├ .gitignore
├ package.json
├ tsconfig.json
├ src/
└ docs/
```
