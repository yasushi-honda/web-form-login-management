# 002-b: loginUser() 実装

## 概要
`loginUser(id: string, password: string)` 関数を実装し、ユーザー認証を行う機能を提供します。

## 要件
- ID/PW 照合ロジック
- セッションID生成
- プロパティサービス保存
- 認証結果返却（成功/失敗）

- [ ] ID/PW 照合実装

- [ ] プロパティサービス保存

- [ ] 単体テスト作成

## 完了条件
- 正しい ID/PW で呼ばれるとセッションID が返却され、間違いの場合はエラーが返ること
