# 003: テンプレートフォーム自動生成機能

## 概要
テンプレートフォームを複製し、ユーザー専用フォームを自動生成する機能を実装します。

## 要件
- `generateUserForm(templateId: string, userId: string)` 関数実装
  - 指定テンプレートを複製
  - 複製先のフォームIDをシートに記録
  - 編集権限をユーザー（Googleアカウント）に付与
- エラー時の削除・ロールバック
